# docker-redash
redash on docker

***DEMO:***


## Usage
```
docker-compose run --rm server create_db
docker-compose up -d
```

## Installation

    $ git clone https://github.com/radioberry/docker-redash.git
    $ cd docker-redash
    $ git clone https://github.com/getredash/redash-amazon-athena-proxy.git
